Project Title: Smart Traffic Intersection Simulator
By Amy Castillo, Daren Xu, Adam Moffat

Project overview:

In this project we will be implementing a Traffic Intersection simulation. This simulation will simulate cars at a two way stree where there is a traffic light. The cars will stop and crete a line when the light is red, will slow down and stop when the light turns yellow and will drive through the light if it is green. Additonally if there is an emergecy vehicle, it will be able to go through any type of light as well as skip the line of vehicles.

Main functionalities:
There will be functions involving the vehicles, traffic lights, lanes, interactions between the vehicles, traffic lights and lanes and a clock.

Project goals:
The overall goal of this project is to be able to properly simulate a basic intersection where vehicles will line up when the lights are red and yellow or continue to drive when the light is green. Additonally we hope to implement a properly work emergncy vehicle which will be able to drive through any light color. The team also is looking forward to being able to learn and utilize simulation applications to be able to simulate our code. The team is also utilizing this project to help deepen understadning of being a C++ programmer as well as deepening our understanding of OOP concepts.
